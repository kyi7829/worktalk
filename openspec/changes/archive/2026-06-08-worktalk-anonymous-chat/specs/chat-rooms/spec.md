## ADDED Requirements

### Requirement: 공개 채팅방 접속
시스템은 URL로 사이트에 접속한 사용자를 별도 인증 없이 즉시 공개 채팅방에 참여시켜야 한다(SHALL). 공개 채팅방은 누구나 접근 가능해야 한다(MUST).

#### Scenario: 공개 채팅방 즉시 입장
- **WHEN** 사용자가 사이트 메인 URL에 접속한다
- **THEN** 회원가입이나 로그인 없이 즉시 공개 채팅방에 입장된다

#### Scenario: 공개 채팅방에서 다른 사용자 메시지 확인
- **WHEN** 공개 채팅방에 여러 사용자가 접속해 있다
- **THEN** 모든 사용자의 메시지가 화면에 표시된다

### Requirement: 비공개 채팅방 생성
시스템은 사용자가 비공개 채팅방을 생성할 수 있게 해야 한다(SHALL). 생성 시 방 유지 시간을 설정할 수 있어야 하며(MUST), 고유한 초대코드가 발급되어야 한다(MUST).

#### Scenario: 비공개 방 생성
- **WHEN** 사용자가 방 유지 시간(예: 1시간, 4시간, 24시간)을 선택하고 방 만들기를 요청한다
- **THEN** 6자리 초대코드가 발급되고 해당 비공개 방으로 입장된다

#### Scenario: 방 만료 시 자동 삭제
- **WHEN** 비공개 방의 설정 유지 시간이 경과한다
- **THEN** 방과 해당 방의 모든 메시지가 삭제되며 더 이상 접속 불가 상태가 된다

### Requirement: 초대코드로 비공개 방 입장
시스템은 사용자가 유효한 초대코드를 입력하면 해당 비공개 채팅방에 입장할 수 있게 해야 한다(SHALL).

#### Scenario: 유효한 초대코드로 입장
- **WHEN** 사용자가 유효한 초대코드를 입력한다
- **THEN** 해당 비공개 채팅방에 입장된다

#### Scenario: 유효하지 않은 초대코드 입력
- **WHEN** 사용자가 존재하지 않거나 만료된 초대코드를 입력한다
- **THEN** 입장이 거부되고 오류 메시지가 표시된다

#### Scenario: 만료된 방의 초대코드 입력
- **WHEN** 사용자가 이미 만료된 방의 초대코드를 입력한다
- **THEN** 방이 만료됐다는 안내 메시지가 표시된다
